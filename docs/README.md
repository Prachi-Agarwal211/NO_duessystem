## Getting Started

1. Clone the repository
2. Copy `.env.example` to `.env.local`
3. Fill in Supabase credentials
4. Run `npm install`
5. Run `npm run dev`

For full setup instructions, see [Deployment Guide](DEPLOYMENT.md).
